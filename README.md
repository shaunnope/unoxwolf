# ONUWolf Telegram Bot
ONUWolf is a Telegram bot to moderate One Night Ultimate Werewolf games. It is currently in development.

## Requirements
- `.env` file with Telegram bot token `BOT_TOKEN` and Firebase config `FB_CONFIG`
